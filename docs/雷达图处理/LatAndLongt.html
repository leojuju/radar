LatAndLongt

简述

用来描述经纬度的类,属性包含经度和纬度,采用角度形式储存。

属性

public double latitude:纬度。

public double longtitude:经度。

LatAndLongt

参数:

Vector v:地心到某点的向量。(所采用坐标系参照直角坐标系)

返回值:

构造函数无返回值。

描述:

排除0向量和v为null的情况。
